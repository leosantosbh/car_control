# Controle de Carro

Instalação:

  - clone do repositório
  
  - cd backend
  - yarn install ou npm install
  
  - cd frontend
  - yarn install ou npm install
  
  
Execução:

  - cd backend
  - yarn install ou npm install
  
  - cd frontend
  - yarn install ou npm install
  

- acessar http://localhost:3000
